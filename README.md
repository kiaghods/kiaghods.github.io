# Personal Website of Kia Ghods

Welcome to the repository for my personal website, hosted at [kiaghods.github.io](https://kiaghods.github.io). This website serves as a platform to showcase my academic work, projects, and interests in Math and Computer Science.

## Features

- **Responsive Web Design**: Crafted to ensure a smooth experience across various devices.
- **Dynamic p5.js Animation**: The website features a captivating background animation built with p5.js. The animation is inspired by [this p5.js sketch](https://editor.p5js.org/wwlr9/sketches/DsSXmS8S9).
- **Project Portfolio**: Detailed showcase of my projects, including research papers, presentations, and technical skills.
- **Contact and Social Links**: Easy access to my professional contact details and social media profiles.

## Website Structure

- `index.html`: Main HTML file for the website.
- `css/style.css`: Custom CSS styles for the website's layout and design.
- `js/`: JavaScript files, including the p5.js animation script.
- `images/`: Contains images used on the website.

## Viewing the Website

The website is hosted on GitHub Pages and can be accessed at [kiaghods.github.io](https://kiaghods.github.io).
